# Animal Crossing AR Chessboard

체스보드 위에 동물의 숲 캐릭터 GIF를 올려놓는 AR 프로젝트입니다.

## 데모

![demo](result.gif)

## 주요 기능

- 체스보드 동영상으로 카메라 캘리브레이션
- PnP 알고리즘을 이용한 실시간 카메라 자세 추정
- 캐릭터 GIF가 체스보드 위에 서 있는 것처럼 표시

## 필요 라이브러리 설치

```bash
pip install opencv-python numpy Pillow
```

## 실행 방법

### 1단계: 카메라 캘리브레이션

```bash
python calibration.py
```

`chessboard.mp4`를 읽어서 캘리브레이션 결과를 `calibration_result.npz`에 저장합니다.

### 2단계: AR 실행

원하는 GIF 파일을 `overlay.gif`로 이름 바꿔서 같은 폴더에 넣고 실행합니다.

```bash
python ar_gif.py
```

## 카메라 자세 추정

매 프레임마다 `cv2.solvePnP()`를 이용해 체스보드 기준 카메라의 자세를 실시간으로 추정합니다.

- **rvec** (회전 벡터): 카메라가 체스보드를 바라보는 방향
- **tvec** (이동 벡터): 체스보드로부터 카메라까지의 거리 및 위치

추정된 rvec, tvec은 GIF를 체스보드 위의 올바른 위치에 투영하는 데 사용됩니다. 카메라 각도가 바뀌어도 GIF가 체스보드에 고정된 것처럼 보이는 이유가 이 때문입니다.

## AR 오브젝트 시각화
단순 좌표축이나 도형 대신 동물의 숲 캐릭터 GIF를 체스보드 위에 세워서 표시합니다.

체스보드 중앙에 수직 평면을 설정하고 GIF 프레임을 투영
cv2.findHomography()로 GIF를 원근감 있게 워핑
GIF 프레임을 순서대로 재생해 캐릭터가 실제로 움직이는 것처럼 표현

## 동작 원리

1. **캘리브레이션** — 체스보드 영상에서 코너를 검출해 카메라 내부 파라미터와 왜곡 계수를 계산
2. **자세 추정** — 매 프레임마다 `cv2.solvePnP()`로 체스보드 기준 카메라의 회전/이동 벡터 추정
3. **GIF 오버레이** — 추정된 자세를 바탕으로 수직 평면을 투영하고, GIF 프레임을 알파 블렌딩으로 합성
